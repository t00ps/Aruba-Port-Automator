# Network Security Automation Tool

## Overview
A Python-based utility designed to automate the verification of network ports on Aruba switches. This tool replaces a time-consuming manual process, significantly reducing authorization time and documentation errors.

## Features
* Establishes secure SSH connections to network equipment.
* Automates the retrieval of switch logs and MAC addresses.
* Ensures strict adherence to company Port Security policies (NAC).

## Tech Stack
* **Language:** Python
* **Libraries:** Netmiko, Paramiko, re, openpyxl
* **Technologies:** Aruba Switches, SSH

## Note
*This repository contains a sanitized version of the script. Hardcoded IPs, credentials, and specific infrastructure details have been removed for security purposes.*
